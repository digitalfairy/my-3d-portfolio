## 💡 Engineering Philosophy

I believe the web should be tactile and memorable. My work focuses on three core pillars:
1.  **Performance:** Optimizing complex 3D scenes for the browser without sacrificing visual quality.
2.  **Responsiveness:** Building layouts that feel custom-tailored whether on a mobile phone or a desktop workstation.
3.  **Scalability:** Using headless CMS and modular component architectures to ensure projects are easy to manage and update.
